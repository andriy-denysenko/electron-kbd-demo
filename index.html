Electron keyboard shortcuts demo

Local shortcuts

Hit Alt+Shift+I on Windows, or Alt+Cmd+I on mac to log a message in the console using local keyboard shortcut.

Global shortcuts

Hit Alt+Ctrl+Q on Windows, or Alt+Cmd+Q on mac to show/hide window.

Hit Ctrl+Q on Windows, or Cmd+Q on mac to exit application.

Browser window shortcuts

Hit Ctrl+I to see a message printed to the console for browser window shortcut. (The next version will write the message on this page)

Pressed keys will be logged in the console. (The next version will log them on this page)
